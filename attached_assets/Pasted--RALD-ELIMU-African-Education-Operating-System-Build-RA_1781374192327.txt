# RALD ELIMU — African Education Operating System

Build RALD ELIMU, a free, modular, cloud-native School Management ERP designed to become the default digital operating system for African education.

ELIMU means "Education."

The platform must serve:

- Nursery Schools
- Primary Schools
- Secondary Schools
- Colleges
- Polytechnics
- Universities
- Vocational Institutions
- Government Education Agencies

ELIMU is part of the RALD Ecosystem and must integrate deeply with:

- RALD Identity
- RALD Mail
- RALD Auth
- RALD ALIA
- SEKANI AI
- WIZMAC
- RALD Voice
- RALD Analytics Cloud

---

# Core Mission

Every African school should be able to go online in under 15 minutes.

No infrastructure expertise required.

No IT department required.

No software licensing fees required.

A school should be able to:

- Register
- Receive a subdomain
- Create staff accounts
- Create student accounts
- Issue school emails
- Manage classes
- Manage attendance
- Manage results

immediately.
