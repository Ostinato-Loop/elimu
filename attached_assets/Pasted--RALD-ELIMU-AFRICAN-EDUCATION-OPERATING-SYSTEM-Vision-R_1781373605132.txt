# RALD ELIMU — AFRICAN EDUCATION OPERATING SYSTEM

## Vision

RALD ELIMU is a free, modular School Management ERP built specifically for Africa.

Its mission is to become the default operating system powering educational institutions across Africa, from kindergarten to university.

ELIMU means "Education."

The platform is designed to eliminate fragmented school systems by providing a unified cloud infrastructure that manages academics, administration, finance, communication, identity, and student success in a single platform.

---

# Core Philosophy

Schools should never be blocked from digitization because of software costs.

The core ELIMU platform is free forever.

Revenue is generated through:

- Transaction infrastructure
